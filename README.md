# [ijquenta.dev](https://ijquenta.dev)

Personal portfolio website built with **Next.js 16**, **Tailwind CSS v4**, and **shadcn/ui**.

## Stack

- Next.js 16 (App Router)
- Tailwind CSS v4
- shadcn/ui
- MDX
- TypeScript
- pnpm

## Development

```bash
pnpm dev
pnpm build
pnpm lint
```
